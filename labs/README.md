## 🔬 Lab 1: AI Assistant-User Conversations
**Focus:** Performance Benchmarking & LLM Comparison.

### 📋 Description
In this lab, I conducted a side-by-side performance evaluation by prompting two AI models simultaneously with identical inputs to measure efficiency and output quality.

### 📊 Key Performance Metrics Measured
* **Response Latency:** Exact time in seconds from prompt to completion.
* **Token Consumption:** Comparative analysis of tokens used per response.
* **Efficiency vs. Accuracy:** Evaluation of response speed versus the depth and correctness of the answer.
